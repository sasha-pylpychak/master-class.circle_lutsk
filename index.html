Базова HTML розмітка

Перша стрічка вказує в якому форматі створена сторінка.

Тег <html> вказує початок і кінець HTML сторінки.

В тег <head> добавляється розмітка для пошукових систем, інформація для браузера користувача, наприклад тег <meta> у приведеному нижче коді задає браузеру кодування яке використовувати для відображення сторінки.

Тег <title> обовязковий тег, задає назву сторінки.

В тег <body> додається вся розмітка потрібна для відображення контенту

1. Створіть папку проекту, і у ній новий текстовий документ (який треба зберегти як index.html) *.html - обов’язково
Скопіюйте та додайте це до вашого файлу.

<!DOCTYPE html>

Для цього нам знадобляться файли зі стилями freelancer.min.css,magnific-popup.css,та папка зі скриптами js.zip

Кінцева структура папки проекту

Підключення файлів стилів плагіна відповідно додати вище тегу <body>
<link href="css/freelancer.min.css" rel="stylesheet">
<link href="css/magnific-popup.css" rel="stylesheet">
